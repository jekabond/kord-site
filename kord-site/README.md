# КОРД — Сайт підрозділу

React-проект сайту КОРД, ГУР МО України.

## 🚀 Запуск

### Вимоги
- Node.js 16+ 
- npm або yarn

### Встановлення залежностей
```bash
npm install
```

### Запуск у режимі розробки
```bash
npm start
```
Відкриється на http://localhost:3000

### Збірка для продакшну
```bash
npm run build
```

## 🎨 Кольори

| Змінна          | Значення   | Використання         |
|-----------------|------------|----------------------|
| `--accent`      | `#c8f000`  | Акцент, CTA-кнопки   |
| `--black`       | `#0a0a0a`  | Фон сторінок         |
| `--dark`        | `#111111`  | Фон секцій           |
| `--dark2`       | `#1a1a1a`  | Картки               |
| `--white`       | `#ffffff`  | Основний текст       |
| `--gray`        | `#888888`  | Другорядний текст    |

## ✏️ Редагування

Кожна секція — окремий компонент. Щоб змінити:
- **Текст/дані** — редагуйте масиви у `.js` файлах компонентів
- **Стилі** — змінюйте відповідний `.css` файл або `--змінні` у `index.css`
- **Фото** — замініть URL у `background` або `<img src=...>` на власні зображення
